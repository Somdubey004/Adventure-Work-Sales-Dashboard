# Adventure Works Sales Dashboard

## Project Overview

This project is an interactive sales dashboard built in Microsoft Excel using the Adventure Works dataset. The dashboard was created to analyze sales performance, revenue trends, profit distribution, customer transactions, and product category performance through dynamic visualizations and business-focused insights.

This project was created as part of my data analysis learning journey to improve my skills in dashboard design, business analysis, interactivity, and data storytelling using Excel.

---

## Objectives

* Analyze sales and profit performance across different years and months
* Track revenue, profit, and transaction trends dynamically
* Create an interactive and user-friendly dashboard experience
* Improve business storytelling using KPIs and visual analytics
* Practice dashboard engineering and Excel-based reporting techniques

---

## Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* VBA (for navigation and interactivity)
* Dynamic Charts
* Form Controls (Option Buttons)

---

## Dashboard Features

* Interactive KPI Dashboard
* Dynamic Yearly Trend Analysis
* Dynamic Monthly Trend Analysis
* Revenue, Profit, and Transaction Switching using Option Buttons
* Time Series Analysis
* Product & Category Analysis
* Sales Performance Dashboard
* Country, Month and Year Filtering using Slicers
* Navigation Buttons with Active Page Highlighting
* Business Insight Storytelling

---

## Key Insights

* Revenue and transaction volume showed strong growth across the selected years.
* Monthly trend analysis helped identify periods of high and low business activity.
* Dynamic charts improved comparison between revenue, profit, and transactions without overcrowding the dashboard.
* Product and category analysis provided insights into top-performing business segments.
* Interactive filters and navigation improved dashboard usability and reporting experience.
